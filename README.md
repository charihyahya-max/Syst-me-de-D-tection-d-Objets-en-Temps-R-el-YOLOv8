 YOLOv8 Object Detection Dashboard (Simulation)

 Overview
This project is a **simulation of a real-time object detection system inspired by YOLOv8**.  
It demonstrates how modern AI vision systems detect, classify, and evaluate objects in an image scene.

The goal of this project is to showcase **computer vision concepts, performance metrics, and visualization techniques** using Python.

---

 Technologies Used
- Python 
- NumPy
- Matplotlib

---

 Key Features
 Synthetic scene generation (urban environment)
 Bounding box visualization for detected objects
 Multi-class object detection simulation
 Performance evaluation:
  - mAP@0.5
  - Precision
  - Recall
  - F1-score
   Training curves (loss & mAP evolution)
   Confusion matrix visualization
   Realistic dashboard-style layout

---

 Metrics Included
- Mean Average Precision (mAP@0.5)
- Precision
- Recall
- F1 Score
- Per-class AP evaluation
